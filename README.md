<h1 style="color: red">list of sway keybinds</h1>
<break>

```bash
Mod + 1 -- 0 -> Switch window
Mod + +/- -> increase/decrease brightness
Mod + "Return or Enter" -> open rofi app menu
Mod + "Q" -> quit/kill program that highlighted.
Mod + "c" -> pause media
Mod + "b" -> toggle next media

```
